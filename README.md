# Basic Redux Boilerplate
A basic React Redux boilerplate

## Installation
```
$ git clone https://github.com/SuperCoac/basic-redux-boilerplate.git
$ cd basic-redux-boilerplate
$ npm install
```

## Development
```
$ npm run dev
$ open http://localhost:8080/
```
Running the webpack-dev-server.

## Test
```
$ npm run test
```
